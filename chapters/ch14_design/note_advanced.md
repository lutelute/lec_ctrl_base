# 第14章 制御系の設計 - 理解者向け編

## 1. 根軌跡法の詳細理論 (Root Locus Method - Advanced Theory)

### 1.1 根軌跡法の数学的基礎

中級編で学んだ根軌跡法の基礎をさらに深め、より詳細な理論を学びます。根軌跡法は、フィードバック制御系の閉ループ極がゲインの変化に対してどのように移動するかを図示する強力な手法です。

閉ループ系の特性方程式は次のように表されます：

$$1 + K \frac{N(s)}{D(s)} = 0$$

ここで、$G(s) = \frac{N(s)}{D(s)}$ は開ループ伝達関数、$K$ は調整可能なゲインパラメータです。

これを変形すると：

$$K \frac{N(s)}{D(s)} = -1$$

この式を満たす複素数 $s$ の軌跡が根軌跡です。

### 1.2 振幅条件と位相条件

根軌跡上の点は、以下の2つの条件を満たす必要があります：

**振幅条件 (Magnitude Condition)**:
$$\left| K \frac{N(s)}{D(s)} \right| = 1$$

すなわち：
$$K = \frac{|D(s)|}{|N(s)|}$$

**位相条件 (Angle Condition)**:
$$\angle N(s) - \angle D(s) = 180° + 360° n \quad (n = 0, \pm 1, \pm 2, \ldots)$$

位相条件は、ある点 $s$ が根軌跡上にあるかどうかを判定するために使用されます。振幅条件は、その点に対応するゲイン $K$ の値を求めるために使用されます。

**例**:
- $G(s) = \frac{1}{s(s+1)}$ の場合
- 点 $s = -0.5 + j0.5$ が根軌跡上にあるか確認するには
- $\angle N(s) = 0°$（零点なし）
- $\angle D(s) = \angle(s) + \angle(s+1) = 135° + 116.6° = 251.6°$
- $\angle N(s) - \angle D(s) = -251.6° \equiv 108.4° \neq 180°$
- したがって、この点は根軌跡上にない

### 1.3 根軌跡の分枝数と対称性

開ループ伝達関数が $n$ 個の極と $m$ 個の零点を持つとき（通常 $n \geq m$）：

1. **分枝数**: $n$ 本の根軌跡が存在
2. **対称性**: 実数係数システムでは、根軌跡は実軸に関して対称
3. **零点への終点**: $m$ 本は有限零点へ、残り $n-m$ 本は無限遠へ

この性質は、システムの次数保存則に基づいています。閉ループ系の特性方程式は $n$ 次であるため、常に $n$ 個の極が存在します。

## 2. 根軌跡の作図規則 (Root Locus Plotting Rules)

### 2.1 基本規則（規則1〜5）

根軌跡を正確に描くための体系的な規則を学びます。

**規則1: 根軌跡の数と対称性**
- 根軌跡は $n$ 本存在（極の数）
- 実軸に関して対称

**規則2: 起点と終点**
- $K = 0$: 根軌跡は開ループの極から始まる（起点）
- $K \to \infty$: 根軌跡は開ループの零点（$m$ 個）または無限遠点（$n-m$ 本）へ向かう（終点）

**規則3: 実軸上の根軌跡**

実軸上のある点が根軌跡上にあるための条件は：

> その点の右側に存在する実極と実零点の総数が奇数である

**例**:
- $G(s) = \frac{1}{s(s+2)(s+4)}$ の場合
- 極：$s = 0, -2, -4$
- 実軸上の区間を調べる：
  - $s > 0$: 右側に極0個 → 偶数 → 根軌跡なし
  - $-2 < s < 0$: 右側に極1個 → 奇数 → 根軌跡あり
  - $-4 < s < -2$: 右側に極2個 → 偶数 → 根軌跡なし
  - $s < -4$: 右側に極3個 → 奇数 → 根軌跡あり

**規則4: 漸近線**

$n > m$ のとき、$n - m$ 本の根軌跡は無限遠へ向かいます。これらの漸近線は：

**漸近線の角度**:
$$\phi_k = \frac{180° + 360° k}{n - m} \quad (k = 0, 1, 2, \ldots, n-m-1)$$

**漸近線の交点（重心）**:
$$\sigma_a = \frac{\sum \text{極} - \sum \text{零点}}{n - m}$$

**例**:
- $G(s) = \frac{1}{s(s+2)(s+4)}$ （$n=3, m=0$）
- 漸近線の数：$3 - 0 = 3$ 本
- 角度：$k=0: 60°$, $k=1: 180°$, $k=2: 300°$
- 重心：$\sigma_a = \frac{(0 + (-2) + (-4)) - 0}{3} = -2$

**規則5: 実軸上の分岐点と接合点**

分岐点（breakaway point）は、根軌跡が実軸から離れる点です。接合点（break-in point）は、根軌跡が実軸に入る点です。

これらの点は、次の条件を満たします：

$$\frac{dK}{ds} = 0$$

または同等に：

$$\sum_{i=1}^{n} \frac{1}{s - p_i} = \sum_{j=1}^{m} \frac{1}{s - z_j}$$

ここで、$p_i$ は極、$z_j$ は零点です。

**例**:
- $G(s) = \frac{1}{s(s+2)}$
- 特性方程式：$s^2 + 2s + K = 0$
- $K = -s^2 - 2s$
- $\frac{dK}{ds} = -2s - 2 = 0$
- $s = -1$（分岐点）
- この点で、2つの実極が衝突して複素共役極となる

### 2.2 詳細規則（規則6〜9）

**規則6: 虚軸との交点**

根軌跡が虚軸と交わる点は、システムの安定限界を表します。この点を求めるには：

1. 特性方程式に $s = j\omega$ を代入
2. 実部と虚部をそれぞれゼロとおく
3. $\omega$ と $K$ を求める

**例**:
- $G(s) = \frac{1}{s(s+1)(s+2)}$
- 特性方程式：$s^3 + 3s^2 + 2s + K = 0$
- $s = j\omega$ を代入：$(j\omega)^3 + 3(j\omega)^2 + 2(j\omega) + K = 0$
- $-j\omega^3 - 3\omega^2 + 2j\omega + K = 0$
- 実部：$-3\omega^2 + K = 0$
- 虚部：$-\omega^3 + 2\omega = 0$
- 虚部から：$\omega(\omega^2 - 2) = 0$ → $\omega = \pm\sqrt{2}$
- 実部から：$K = 3\omega^2 = 6$
- 交点：$s = \pm j\sqrt{2}$、限界ゲイン：$K = 6$

**規則7: 出発角と到着角**

複素極や複素零点からの根軌跡の出発角（departure angle）や到着角（arrival angle）は、位相条件から求められます。

**複素極 $p_k$ からの出発角 $\theta_d$**:
$$\theta_d = 180° + \sum \angle(p_k - z_j) - \sum_{i \neq k} \angle(p_k - p_i)$$

**複素零点 $z_k$ への到着角 $\theta_a$**:
$$\theta_a = 180° - \sum_{j \neq k} \angle(z_k - z_j) + \sum \angle(z_k - p_i)$$

**例**:
- $G(s) = \frac{1}{(s+1-j)(s+1+j)}$
- 複素極：$p_1 = -1 + j$, $p_2 = -1 - j$
- $p_1$ からの出発角を求める：
- $\angle(p_1 - p_2) = \angle(2j) = 90°$
- $\theta_d = 180° - 90° = 90°$
- したがって、$p_1$ から真上（虚軸の正方向）へ出発する

**規則8: 根軌跡と等減衰比線**

制御系の設計では、所望の減衰比 $\zeta$ を持つ極の位置を求めることがよくあります。減衰比が一定の点の軌跡は、原点を中心とする角度 $\theta = \arccos(\zeta)$ の直線となります。

$$\zeta = \cos \theta$$

**例**:
- 減衰比 $\zeta = 0.5$ が必要な場合
- $\theta = \arccos(0.5) = 60°$
- 実軸の負の方向から $60°$ と $120°$ の直線上に極を配置する

**規則9: 根軌跡の分離と再結合**

複数の根軌跡が実軸上で衝突して分離したり、複素平面から実軸へ再結合したりする点は、規則5の分岐点・接合点として求められます。

### 2.3 根軌跡作図の実践例

**例題**: 次の伝達関数の根軌跡を描きなさい。

$$G(s) = \frac{s + 2}{s(s+1)(s+3)}$$

**解**:

1. **極と零点**:
   - 極：$p_1 = 0$, $p_2 = -1$, $p_3 = -3$
   - 零点：$z_1 = -2$

2. **規則1**: 3本の根軌跡、実軸対称

3. **規則2**:
   - 起点：$s = 0, -1, -3$
   - 終点：$s = -2$（1本）、無限遠（2本）

4. **規則3**: 実軸上の根軌跡
   - $-1 < s < 0$: 右側に極1個、零点0個 → 奇数 → 根軌跡あり
   - $-2 < s < -1$: 右側に極1個、零点0個 → 奇数 → 根軌跡あり
   - $-3 < s < -2$: 右側に極2個、零点1個 → 奇数 → 根軌跡あり
   - $s < -3$: 右側に極3個、零点1個 → 偶数 → 根軌跡なし

5. **規則4**: 漸近線（$n-m = 3-1 = 2$ 本）
   - 角度：$\phi_0 = 90°$, $\phi_1 = 270°$
   - 重心：$\sigma_a = \frac{(0-1-3) - (-2)}{2} = \frac{-2}{2} = -1$

6. **規則5**: 分岐点
   - $\frac{1}{s} + \frac{1}{s+1} + \frac{1}{s+3} = \frac{1}{s+2}$
   - 計算により、$s \approx -0.42$ と $s \approx -2.58$ に分岐点

この情報から、根軌跡の全体像を描くことができます。

## 3. 根軌跡を用いたゲイン設計 (Gain Design using Root Locus)

### 3.1 仕様から極配置への変換

制御系の設計では、時間領域の仕様（整定時間、オーバーシュートなど）が与えられることが多いです。これらを極の配置に変換する必要があります。

**整定時間 (Settling Time) $T_s$**:

2%規範で考えると：
$$T_s \approx \frac{4}{\zeta \omega_n}$$

主極が実軸上にある場合：
$$T_s \approx \frac{4}{|\text{Re}(s)|}$$

**オーバーシュート (Overshoot) $M_p$**:

$$M_p = e^{-\frac{\pi \zeta}{\sqrt{1-\zeta^2}}} \times 100\%$$

この式から減衰比 $\zeta$ を求めると：

$$\zeta = \frac{-\ln(M_p/100)}{\sqrt{\pi^2 + \ln^2(M_p/100)}}$$

**例**:
- 要求仕様：$T_s \leq 2$ [s]、$M_p \leq 10\%$
- オーバーシュートから：$\zeta \geq 0.59$
- 整定時間から：$\zeta \omega_n \geq 2$ → $\omega_n \geq 3.39$
- これより、極の実部は $\text{Re}(s) \leq -2$、減衰比 $\zeta \geq 0.59$

### 3.2 根軌跡上での極配置

根軌跡を描いた後、所望の極位置を実現するゲイン $K$ を求めます。

**手順**:

1. **所望の極の領域を決定**: 仕様から導かれる制約（減衰比、整定時間など）を複素平面上に図示
2. **根軌跡との交点を探す**: 根軌跡がこの領域を通る点を見つける
3. **ゲインを計算**: 振幅条件からゲイン $K$ を求める

$$K = \left| \frac{D(s)}{N(s)} \right|_{s=s_d}$$

ここで、$s_d$ は所望の極位置です。

**例**:
- $G(s) = \frac{1}{s(s+2)}$
- 所望の極：$s = -1 \pm j$ （$\zeta = 0.707$）
- 振幅条件：$K = |s(s+2)|_{s=-1+j} = |(-1+j)(-1+j+2)| = |(-1+j)(1+j)| = |-2j| = 2$

### 3.3 主極と副極の考え方

高次システムでは、すべての極を直接制御することは困難です。**主極 (dominant poles)** と **副極 (secondary poles)** の概念が重要です。

**主極**:
- 虚軸に最も近い極（実部が最も小さい極）
- 過渡応答を主に決定する
- 設計の主な対象

**副極**:
- 虚軸から遠い極（実部が大きい極）
- 応答への影響が小さい
- 主極の実部の5〜10倍以上離れていれば無視可能

**例**:
- 極：$s_1 = -1 + j2$, $s_2 = -1 - j2$, $s_3 = -10$
- $s_1, s_2$ が主極（虚軸に近い）
- $s_3$ は副極（$|s_3|/|s_1| \approx 4.5$ で影響は小さい）

### 3.4 零点の影響

開ループ零点は、根軌跡の形状と過渡応答に大きな影響を与えます。

**零点が極に近い場合**:
- 零点が主極に近いと、オーバーシュートが増加
- 右半平面の零点（非最小位相零点）は、逆応答（undershoot）を引き起こす可能性

**零点配置の指針**:
- 零点は主極から十分離す（実部で5倍以上）
- 可能な限り虚軸から遠ざける

## 4. 位相進み・遅れ補償器 (Lead-Lag Compensators)

### 4.1 補償器の必要性

単純なゲイン調整だけでは、すべての設計仕様を満たせない場合があります。このとき、**補償器 (compensator)** を用いて根軌跡の形状を変更します。

主な補償器の種類：
1. **位相進み補償器 (lead compensator)**: 過渡応答の改善
2. **位相遅れ補償器 (lag compensator)**: 定常特性の改善
3. **位相進み遅れ補償器 (lead-lag compensator)**: 両方の改善

### 4.2 位相進み補償器の設計

**位相進み補償器の伝達関数**:

$$C(s) = K_c \frac{s + z_c}{s + p_c} \quad (z_c < p_c)$$

または、標準形で：

$$C(s) = K_c \alpha \frac{Ts + 1}{\alpha Ts + 1} \quad (\alpha > 1)$$

ここで、
- $z_c = -1/T$: 零点
- $p_c = -1/(\alpha T) = -\alpha/T$: 極
- $\alpha$: 減衰定数（通常 $\alpha = 5 \sim 15$）

**位相進み補償器の効果**:
- 高周波帯域で位相を進める（最大 $\phi_m = \arcsin\frac{\alpha-1}{\alpha+1}$）
- 根軌跡を左方向へ引き寄せる
- 応答速度の向上、安定余裕の増加

**設計手順**:

1. **所望の極位置を決定**: 仕様から要求される $s_d$
2. **不足位相を計算**: $s_d$ における開ループの位相 $\angle G(s_d)$ を計算
3. **必要な位相進み角を求める**: $\phi_m = 180° - |\angle G(s_d)| + \text{余裕}$（通常5〜12°の余裕を追加）
4. **$\alpha$ を決定**: $\sin \phi_m = \frac{\alpha - 1}{\alpha + 1}$ より $\alpha = \frac{1 + \sin \phi_m}{1 - \sin \phi_m}$
5. **零点と極を配置**: 最大位相進み角が $s_d$ で生じるように配置
   - $z_c p_c = \omega_d^2$（$\omega_d$ は $s_d$ の虚部）
   - $z_c = \frac{\omega_d}{\sqrt{\alpha}}$, $p_c = \omega_d \sqrt{\alpha}$
6. **ゲイン $K_c$ を調整**: $s_d$ が根軌跡上にあるように

**例**:
- $G(s) = \frac{1}{s(s+1)}$
- 所望の極：$s_d = -2 + j2$（$\zeta = 0.707$）
- 補償なしの場合、$s_d$ は根軌跡上にない
- $s_d$ での開ループ位相：$\angle G(s_d) \approx -227°$
- 必要な位相進み：$\phi_m = 180° - 227° + 10° = -37°$ → 約40°必要
- $\alpha = \frac{1 + \sin 40°}{1 - \sin 40°} \approx 3.4$
- 零点：$z_c \approx 1.2$、極：$p_c \approx 4.0$
- 補償器：$C(s) = K_c \frac{s + 1.2}{s + 4.0}$

### 4.3 位相遅れ補償器の設計

**位相遅れ補償器の伝達関数**:

$$C(s) = K_c \frac{s + z_c}{s + p_c} \quad (z_c > p_c)$$

または、標準形で：

$$C(s) = K_c \frac{Ts + 1}{\beta Ts + 1} \quad (\beta > 1)$$

ここで、
- $z_c = -1/T$: 零点
- $p_c = -1/(\beta T) = -1/(T\beta)$: 極
- $\beta$: 減衰定数（通常 $\beta = 5 \sim 20$）

**位相遅れ補償器の効果**:
- 低周波帯域でゲインを増加（定常ゲインが $\beta$ 倍）
- 過渡応答にはほとんど影響しない（零点と極を左半平面の原点近くに配置するため）
- 定常偏差の改善

**設計手順**:

1. **無補償系で所望の極配置を実現するゲイン $K_0$ を求める**
2. **定常偏差の要求から $\beta$ を決定**: 定常ゲインを $\beta$ 倍にする必要
3. **零点と極を配置**: 極を原点の近くに（$p_c \approx 0.01 \sim 0.1$）
   - $z_c = \beta p_c$
4. **ゲイン調整**: 所望の極配置を維持するように $K_c$ を調整

**例**:
- $G(s) = \frac{1}{s(s+1)(s+5)}$
- 所望の極：P制御で $K = 10$ のとき達成
- しかし定常偏差が大きい（位置偏差定数 $K_p = 2$）
- $K_p$ を10にしたい → $\beta = 5$ 必要
- 補償器：$C(s) = \frac{s + 0.5}{s + 0.1}$（$\beta = 5$）
- これにより、過渡応答を維持したまま定常偏差を1/5に削減

### 4.4 位相進み遅れ補償器

位相進み補償と位相遅れ補償の両方の利点を組み合わせた補償器です。

**伝達関数**:

$$C(s) = K_c \frac{(s + z_{c1})(s + z_{c2})}{(s + p_{c1})(s + p_{c2})}$$

ここで、
- $z_{c1} < p_{c1}$: 位相進み部分
- $z_{c2} > p_{c2}$: 位相遅れ部分

**設計手順**:
1. 位相進み部分を設計（過渡応答の改善）
2. 位相遅れ部分を設計（定常特性の改善）
3. 両者を組み合わせて全体の補償器を構成

### 4.5 補償器選択の指針

| 目的 | 補償器 | 特徴 |
|------|--------|------|
| 応答速度向上 | 位相進み | 帯域幅拡大、安定余裕増加 |
| 定常偏差削減 | 位相遅れ | 低周波ゲイン増加、応答への影響小 |
| 両方必要 | 位相進み遅れ | 総合的な性能改善 |
| 外乱抑制 | 位相遅れ | 低周波の外乱に有効 |

## 5. 設計最適化の考え方 (Design Optimization)

### 5.1 多目的設計問題

実際の制御系設計では、複数の相反する目的を同時に達成する必要があります：

1. **応答速度**: 速いほど良い
2. **オーバーシュート**: 小さいほど良い
3. **定常偏差**: 小さいほど良い
4. **制御入力**: 小さいほど良い（アクチュエータの制約）
5. **ロバスト性**: 外乱やモデル誤差に強い

これらの目的は、しばしばトレードオフの関係にあります。

### 5.2 性能指標

制御系の性能を定量的に評価するための指標：

**積分型性能指標**:

1. **IAE (Integral of Absolute Error)**:
   $$J_{IAE} = \int_0^\infty |e(t)| dt$$

2. **ISE (Integral of Squared Error)**:
   $$J_{ISE} = \int_0^\infty e^2(t) dt$$

3. **ITAE (Integral of Time-weighted Absolute Error)**:
   $$J_{ITAE} = \int_0^\infty t |e(t)| dt$$

4. **ITSE (Integral of Time-weighted Squared Error)**:
   $$J_{ITSE} = \int_0^\infty t e^2(t) dt$$

**特徴**:
- ISE: 大きな偏差を重視、計算が容易
- IAE: 偏差を均等に評価
- ITAE: 整定時間を重視（時間重み）
- ITSE: 両方の特性

### 5.3 最適ゲイン設計

性能指標を最小化するゲインを求める問題は、最適制御問題の一種です。

**2次形式評価関数**:

$$J = \int_0^\infty (e^2(t) + \rho u^2(t)) dt$$

ここで、$\rho$ は制御入力の重み係数です。$\rho$ を変化させることで、追従性能と制御コストのバランスを調整できます。

- $\rho$ 小: 制御入力を惜しまず、高速応答を優先
- $\rho$ 大: 制御入力を節約、穏やかな応答

### 5.4 ロバスト安定性

実システムでは、モデル化誤差や外乱が存在します。これらに対する**ロバスト性 (robustness)** が重要です。

**ゲイン余裕 (Gain Margin) $GM$**:
- システムが不安定になる限界のゲイン変動
- $GM > 6$ dB（2倍）が望ましい

**位相余裕 (Phase Margin) $PM$**:
- システムが不安定になる限界の位相変動
- $PM > 30°$〜$60°$ が望ましい

根軌跡設計では：
- 極を虚軸から十分離す（実部を大きく）
- 安定余裕を確保した極配置を選ぶ

### 5.5 極配置の実践的指針

経験的に良好な性能を与える極配置：

**Butterworth配置**:
- 極を原点中心の円周上に等角度に配置
- フラットな周波数応答
- 適度なオーバーシュート

**Bessel配置**:
- 線形位相特性を持つ配置
- 波形歪みが小さい
- オーバーシュートが小さい

**ITAE最適配置**:
- ITAE評価関数を最小化する極配置
- 標準形の係数が表にまとめられている
- 実用的に優れた応答

**2次系の場合**:
- $\zeta = 0.7$〜$0.8$ が汎用的に良好
- $\zeta = 0.5$: 速い応答、大きなオーバーシュート（約16%）
- $\zeta = 1.0$: 振動なし、遅い応答

### 5.6 設計プロセスの全体像

実際の設計では、以下のような反復的なプロセスを経ます：

1. **仕様の明確化**: 数値目標とその優先順位
2. **初期設計**: 根軌跡法で基本的なゲインを決定
3. **性能評価**: シミュレーションで性能指標を計算
4. **補償器の検討**: 仕様を満たさない場合、補償器を追加
5. **パラメータ調整**: 性能指標を最適化
6. **ロバスト性検証**: パラメータ変動や外乱に対する感度を確認
7. **実機試験**: 実システムで動作確認と微調整

**設計の心得**:
- 完璧な解は存在しない（トレードオフの受容）
- 仕様の優先順位を明確にする
- シンプルな解を優先する（複雑な補償器は実装が困難）
- 安全余裕を十分に確保する
- 実機との整合性を常に確認する

## 6. 高度な設計手法への展開

### 6.1 状態フィードバック制御との関係

根軌跡法は古典制御理論に基づく設計法ですが、現代制御理論の**状態フィードバック制御**と密接に関連しています。

**極配置法 (pole placement method)** では、状態フィードバックゲインを用いて閉ループ極を任意の位置に配置できます。根軌跡法は、この極配置の候補を視覚的に探索する手法と捉えることができます。

### 6.2 周波数応答法との関係

根軌跡法は**s平面（複素平面）**での設計法ですが、**周波数応答法（Bode線図、Nyquist線図）**とも相補的な関係にあります。

- 根軌跡：極の位置から過渡応答を予測
- 周波数応答：周波数特性から定常特性と安定余裕を評価

両方の観点から設計を検証することが望ましいです。

### 6.3 実装上の考慮事項

理論的な設計を実システムに適用する際の注意点：

1. **離散化**: デジタル制御では、連続時間設計を離散化する必要がある
   - 双一次変換、零次ホールド法など
   - サンプリング周期は閉ループ帯域幅の10倍以上が望ましい

2. **微分の実装**: 理想的な微分は実装不可能
   - 実用微分（擬似微分）を使用
   - ローパスフィルタとの組み合わせ

3. **積分飽和（ワインドアップ）**: 制御入力が飽和すると積分項が異常に蓄積
   - アンチワインドアップ機構の実装
   - 積分項のリセットや制限

4. **センサノイズ**: 測定ノイズが微分動作を妨げる
   - フィルタリング
   - 微分ゲインの抑制

### 6.4 まとめの展望

根軌跡法は、制御系設計の強力なツールであり、以下の特徴を持ちます：

**利点**:
- ゲイン変化と極の関係を直感的に理解できる
- 補償器設計の指針が明確
- 手計算でも概略設計が可能

**限界**:
- 1つのパラメータ（ゲイン）の変化しか扱えない
- 多入出力系への拡張が困難
- 零点の影響を完全には表現できない

現代の制御系設計では、根軌跡法を出発点として、数値最適化、ロバスト制御、適応制御などの高度な手法へと発展させることが一般的です。
