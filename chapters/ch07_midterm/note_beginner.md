## 1. 逆ラプラス変換入門 (Introduction to Inverse Laplace Transform)

### 1.1 逆ラプラス変換とは

**逆ラプラス変換 (Inverse Laplace Transform)** は、s領域で表された関数F(s)を時間領域の関数f(t)に変換する操作です。ラプラス変換の逆操作として、制御工学では非常に重要です。

数学的には、以下のように定義されます：

$$f(t) = \mathcal{L}^{-1}\{F(s)\} = \frac{1}{2\pi j}\int_{c-j\infty}^{c+j\infty} F(s)e^{st}ds$$

ただし、実際の計算では、この複素積分を直接計算するのではなく、**変換表 (Transform Table)** を使用することが一般的です。

### 1.2 基本的な変換表

以下は最も基本的な逆ラプラス変換の対応表です：

| F(s) | f(t) | 説明 |
|------|------|------|
| $\frac{1}{s}$ | $1$ | 定数関数 |
| $\frac{1}{s^2}$ | $t$ | ランプ関数 |
| $\frac{1}{s^n}$ | $\frac{t^{n-1}}{(n-1)!}$ | 高次の項 |
| $\frac{1}{s+a}$ | $e^{-at}$ | 指数減衰 |
| $\frac{1}{(s+a)^2}$ | $te^{-at}$ | 減衰ランプ |
| $\frac{\omega}{s^2+\omega^2}$ | $\sin(\omega t)$ | 正弦波 |
| $\frac{s}{s^2+\omega^2}$ | $\cos(\omega t)$ | 余弦波 |

> **注意**: これらの変換は $t \geq 0$ で有効であり、$t < 0$ では $f(t) = 0$ とします（因果性）。

### 1.3 簡単な例題

**例1**: $F(s) = \frac{3}{s}$ の逆ラプラス変換を求めよ。

**解**:
変換表から $\mathcal{L}^{-1}\{\frac{1}{s}\} = 1$ なので、

$$f(t) = \mathcal{L}^{-1}\{\frac{3}{s}\} = 3 \cdot \mathcal{L}^{-1}\{\frac{1}{s}\} = 3$$

**例2**: $F(s) = \frac{1}{s+5}$ の逆ラプラス変換を求めよ。

**解**:
変換表から $\mathcal{L}^{-1}\{\frac{1}{s+a}\} = e^{-at}$ なので、$a=5$ として、

$$f(t) = e^{-5t}$$

**例3**: $F(s) = \frac{2}{s^2+4}$ の逆ラプラス変換を求めよ。

**解**:
$\omega^2 = 4$ より $\omega = 2$ として、

$$F(s) = \frac{2}{s^2+4} = \frac{2}{s^2+2^2} = \frac{2}{2} \cdot \frac{2}{s^2+2^2}$$

変換表から $\mathcal{L}^{-1}\{\frac{\omega}{s^2+\omega^2}\} = \sin(\omega t)$ なので、

$$f(t) = \sin(2t)$$

## 2. インパルス応答と伝達関数 (Impulse Response and Transfer Function)

### 2.1 インパルス応答とは

**インパルス応答 (Impulse Response)** $g(t)$ は、システムに単位インパルス信号 $\delta(t)$ を入力したときの出力応答です。インパルス応答は、システムの動的特性を完全に表現します。

**単位インパルス (Unit Impulse)** $\delta(t)$ は、以下の性質を持つ特殊な関数です：

$$\delta(t) = \begin{cases}
\infty & t = 0 \\
0 & t \neq 0
\end{cases}, \quad \int_{-\infty}^{\infty} \delta(t)dt = 1$$

### 2.2 伝達関数とは

**伝達関数 (Transfer Function)** $G(s)$ は、システムの入力 $U(s)$ と出力 $Y(s)$ のラプラス変換の比として定義されます：

$$G(s) = \frac{Y(s)}{U(s)}$$

ここで重要なのは、**インパルス応答のラプラス変換が伝達関数に等しい**という関係です：

$$G(s) = \mathcal{L}\{g(t)\}$$

これは、インパルス信号のラプラス変換が $\mathcal{L}\{\delta(t)\} = 1$ であることから導かれます。

### 2.3 基本的な導出例

**例1**: インパルス応答が $g(t) = e^{-3t}$ のシステムの伝達関数を求めよ。

**解**:
伝達関数は、インパルス応答のラプラス変換なので、

$$G(s) = \mathcal{L}\{e^{-3t}\} = \frac{1}{s+3}$$

**例2**: 伝達関数が $G(s) = \frac{5}{s+2}$ のシステムのインパルス応答を求めよ。

**解**:
インパルス応答は、伝達関数の逆ラプラス変換なので、

$$g(t) = \mathcal{L}^{-1}\{\frac{5}{s+2}\} = 5e^{-2t}$$

> **ポイント**: 伝達関数を知っていれば、システムの時間応答を逆ラプラス変換で求めることができます。

## 3. フィードバック系の基礎 (Basics of Feedback Systems)

### 3.1 フィードバック制御系とは

**フィードバック制御系 (Feedback Control System)** は、システムの出力を測定し、その情報を入力側にフィードバックすることで、目標値との誤差を減少させる制御方式です。

基本的なフィードバック系のブロック線図は以下のようになります：

```
       +     E(s)      Y(s)
R(s) --->(+)----> G(s) ----> Y(s)
       - ^                   |
         |                   |
         +------- H(s) <-----+
```

ここで、
- $R(s)$: 目標値（参照入力）
- $Y(s)$: 出力
- $E(s)$: 誤差信号 $= R(s) - H(s)Y(s)$
- $G(s)$: フォワードパス伝達関数
- $H(s)$: フィードバックパス伝達関数

### 3.2 閉ループ伝達関数

フィードバック系全体の伝達関数（閉ループ伝達関数、Closed-loop Transfer Function）は、以下の公式で求められます：

$$\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}$$

この式は非常に重要で、**フィードバック系の基本公式**として覚えておく必要があります。

**導出**:
誤差信号は、

$$E(s) = R(s) - H(s)Y(s)$$

出力は、

$$Y(s) = G(s)E(s) = G(s)[R(s) - H(s)Y(s)]$$

これを整理すると、

$$Y(s) = G(s)R(s) - G(s)H(s)Y(s)$$

$$Y(s) + G(s)H(s)Y(s) = G(s)R(s)$$

$$Y(s)[1 + G(s)H(s)] = G(s)R(s)$$

$$\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}$$

### 3.3 簡単な例題

**例1**: $G(s) = \frac{2}{s+1}$、$H(s) = 1$ の負フィードバック系の閉ループ伝達関数を求めよ。

**解**:
公式に代入すると、

$$\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)} = \frac{\frac{2}{s+1}}{1 + \frac{2}{s+1} \cdot 1}$$

$$= \frac{\frac{2}{s+1}}{\frac{s+1+2}{s+1}} = \frac{2}{s+3}$$

**例2**: 上記の閉ループ系に単位ステップ入力 $R(s) = \frac{1}{s}$ を加えたときの出力を求めよ。

**解**:
出力のラプラス変換は、

$$Y(s) = \frac{2}{s+3} \cdot \frac{1}{s} = \frac{2}{s(s+3)}$$

これを部分分数展開すると（詳細は中級編で学びます）、

$$Y(s) = \frac{2/3}{s} - \frac{2/3}{s+3}$$

逆ラプラス変換すると、

$$y(t) = \frac{2}{3}(1 - e^{-3t})$$

> **注意**: $t \to \infty$ で $y(t) \to \frac{2}{3}$ となり、定常値は2/3です。入力の振幅1に対して出力が2/3なので、定常偏差が存在します。

## まとめ

この初級編では、中間試験に向けて以下を学びました：

### 逆ラプラス変換
- 逆ラプラス変換は、s領域の関数F(s)を時間領域の関数f(t)に変換する操作
- 実用上は変換表を使用して計算する
- 基本的な変換ペア（$\frac{1}{s} \leftrightarrow 1$、$\frac{1}{s+a} \leftrightarrow e^{-at}$ など）を覚える

### インパルス応答と伝達関数
- インパルス応答 $g(t)$ は、単位インパルス入力に対するシステムの応答
- 伝達関数 $G(s)$ は、入力と出力のラプラス変換の比 $G(s) = Y(s)/U(s)$
- 重要な関係式：$G(s) = \mathcal{L}\{g(t)\}$（伝達関数はインパルス応答のラプラス変換）

### フィードバック系の基礎
- フィードバック制御系は、出力を測定して入力側にフィードバックする制御方式
- 閉ループ伝達関数の公式：$\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}$
- この公式を使って、フィードバック系全体の応答を求めることができる
