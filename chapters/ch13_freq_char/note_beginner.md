# 第13章 制御系の周波数特性 - 初級編

## 1. 周波数応答入門 (Introduction to Frequency Response)

### 1.1 周波数応答とは何か

**周波数応答 (frequency response)** は、制御システムに正弦波信号を入力したときの出力の振る舞いを表します。システムの伝達関数 $G(s)$ において、$s = j\omega$ として正弦波入力に対する応答を調べます：

$$G(j\omega)$$

ここで：
- $\omega$ は**角周波数 (angular frequency)** [rad/s]
- $j$ は虚数単位 ($j^2 = -1$)
- $G(j\omega)$ は複素数で、**ゲイン（振幅比）** と**位相差**の情報を含みます

**日常例での理解**:

スピーカーシステムを考えてみましょう：
- 低音（低周波）: 重低音がどれくらい増幅されるか
- 中音（中周波）: ボーカルの音量がどうなるか
- 高音（高周波）: シンバルの音がどう響くか

これらは周波数に応じたシステムの「応答」を表しています。

### 1.2 なぜ周波数特性を調べるのか

制御システムの周波数特性を調べることで、以下がわかります：

1. **システムの安定性**: 閉ループ系が安定かどうかを判定できます

2. **過渡応答の予測**: 周波数特性から時間応答の速さや振動の程度を予測できます

3. **ノイズへの影響**: 高周波ノイズをどの程度抑制できるかがわかります

4. **制御器の設計指針**: 希望する周波数特性を得るために制御器をどう設計すべきかがわかります

> **注意**: 周波数応答は、**線形システム**にのみ適用できます。非線形システムでは周波数応答の概念は厳密には使えません。

## 2. ボード線図 (Bode Diagram)

### 2.1 ボード線図とは

**ボード線図 (Bode diagram)** は、周波数応答 $G(j\omega)$ を2つのグラフで表現する方法です：

1. **ゲイン線図 (gain plot)**: 振幅比を周波数の関数として表示
   $$\text{ゲイン [dB]} = 20\log_{10}|G(j\omega)|$$

2. **位相線図 (phase plot)**: 位相差を周波数の関数として表示
   $$\text{位相 [deg]} = \arg(G(j\omega)) \times \frac{180}{\pi}$$

**特徴**:
- 横軸: 周波数 $\omega$ を**対数スケール**で表示
- ゲイン線図の縦軸: ゲイン [dB]
- 位相線図の縦軸: 位相 [度]

### 2.2 ボード線図の読み方

**ゲイン線図の読み方**:
- **0 dB**: 入力と出力の振幅が同じ
- **正のdB値**: 出力が入力より大きい（増幅）
- **負のdB値**: 出力が入力より小さい（減衰）

**位相線図の読み方**:
- **0度**: 入力と出力の位相が同じ
- **負の角度**: 出力が入力より遅れる（位相遅れ）
- **正の角度**: 出力が入力より進む（位相進み）

**例**:
- ゲイン +20 dB → 振幅比は $10^{20/20} = 10$ 倍
- ゲイン -6 dB → 振幅比は $10^{-6/20} \approx 0.5$ 倍
- 位相 -90° → 出力は入力より90度（1/4周期）遅れる

### 2.3 1次遅れ系のボード線図

最も基本的な1次遅れ系の伝達関数：

$$G(s) = \frac{K}{1 + Ts}$$

ここで、$K$ はゲイン、$T$ は時定数です。

周波数応答は：

$$G(j\omega) = \frac{K}{1 + j\omega T}$$

**ゲイン特性**:
- 低周波 ($\omega \ll 1/T$): $|G(j\omega)| \approx K$ → 0 dB付近（$K=1$の場合）
- 高周波 ($\omega \gg 1/T$): $|G(j\omega)| \approx \frac{K}{\omega T}$ → -20 dB/decadeで減衰

**位相特性**:
- 低周波: 位相 ≈ 0°
- $\omega = 1/T$ (折点周波数): 位相 = -45°
- 高周波: 位相 → -90°

> **注意**: 1次遅れ系は、位相が最大でも -90° までしか遅れません。安定性に優れた特性です。

## 3. ナイキスト線図 (Nyquist Diagram)

### 3.1 ナイキスト線図とは

**ナイキスト線図 (Nyquist diagram)** は、周波数応答 $G(j\omega)$ を複素平面上に描いた軌跡です：

- **横軸**: 実部 $\text{Re}[G(j\omega)]$
- **縦軸**: 虚部 $\text{Im}[G(j\omega)]$
- **軌跡**: 周波数 $\omega$ を 0 から $\infty$ まで変化させたときの点の集合

### 3.2 ナイキスト線図の読み方

**基本的な読み方**:
- 各点は特定の周波数 $\omega$ における複素数 $G(j\omega)$ を表します
- 原点からの距離: ゲイン（振幅比）
- 実軸正方向からの角度: 位相

**重要な点**:
- **(-1, 0) 点**: 閉ループ系の安定性を判定する上で最も重要な点
- ナイキスト軌跡がこの点を囲むかどうかで安定性が決まります

### 3.3 ナイキスト線図と安定性

**直感的理解**:

閉ループ系 $\frac{G(s)}{1 + G(s)}$ の安定性を考えます。

- 軌跡が **(-1, 0) 点を囲まない** → **安定**
- 軌跡が **(-1, 0) 点を囲む** → **不安定**
- 軌跡が **(-1, 0) 点を通る** → **臨界安定**

**なぜ (-1, 0) 点が重要なのか**:

閉ループの特性方程式 $1 + G(s) = 0$ より、$G(s) = -1$ となる点が極（不安定の原因）に対応します。複素平面上で $-1$ は点 (-1, 0) です。

**例**:
- 比例ゲイン系 $G(s) = K$: ゲイン $K < 1$ なら、ナイキスト軌跡は実軸上の点で (-1, 0) より右側にあり、安定です

## 4. ゲイン余裕と位相余裕 (Gain and Phase Margins)

### 4.1 安定性余裕とは

**安定性余裕 (stability margin)** は、システムが安定な状態から不安定になるまでの「余裕」を表す指標です。

2つの重要な指標：

1. **ゲイン余裕 (gain margin, GM)**: ゲインをどれだけ増やしても安定を保てるか
2. **位相余裕 (phase margin, PM)**: 位相遅れをどれだけ増やしても安定を保てるか

### 4.2 ゲイン余裕の定義

**ゲイン余裕**は、位相が -180° となる周波数 $\omega_{pc}$（**位相交差周波数**）でのゲインの逆数です：

$$\text{GM [dB]} = -20\log_{10}|G(j\omega_{pc})|$$

**直感的な意味**:
- GM = 20 dB → ゲインを 10 倍にしても安定
- GM = 6 dB → ゲインを 2 倍にしても安定
- GM = 0 dB → 臨界安定（境界線）

**一般的な設計基準**: GM ≥ 6 dB （安全側の設計）

### 4.3 位相余裕の定義

**位相余裕**は、ゲインが 0 dB（振幅比 = 1）となる周波数 $\omega_{gc}$（**ゲイン交差周波数**）での位相と -180° との差です：

$$\text{PM [deg]} = 180° + \arg(G(j\omega_{gc}))$$

**直感的な意味**:
- PM = 60° → 位相を 60° 遅らせても安定
- PM = 30° → 位相を 30° 遅らせても安定
- PM = 0° → 臨界安定（境界線）

**一般的な設計基準**: PM ≥ 30° （より良い応答には PM ≥ 45°）

### 4.4 時間応答との関係

周波数領域の安定性余裕は、時間領域の過渡応答と深い関係があります：

**位相余裕と減衰比の関係**:

$$\text{PM [deg]} \approx 100 \times \zeta$$

ここで、$\zeta$ は減衰比です。

**例**:
- PM = 60° → $\zeta \approx 0.6$ → オーバーシュート約10%
- PM = 45° → $\zeta \approx 0.45$ → オーバーシュート約20%
- PM = 30° → $\zeta \approx 0.3$ → オーバーシュート約35%

> **注意**: これは2次系の近似です。高次系では厳密には成り立ちませんが、設計の目安として有用です。
