# 第3章 LTIシステムの表現 - 初級編

## 1. LTIシステムとは何か (What is an LTI System)

### 1.1 システムの定義

**システム (system)** とは、入力信号を受け取り、何らかの処理を行って出力信号を生成する仕組みのことです。制御工学では、このシステムの振る舞いを理解し、設計することが重要です。

**LTIシステム (Linear Time-Invariant System)** は、以下の2つの重要な性質を持つシステムです：

1. **線形性 (Linearity)**: 入力と出力の関係が線形である
2. **時不変性 (Time Invariance)**: システムの特性が時間によって変化しない

この2つの性質により、LTIシステムは数学的に扱いやすく、実際の多くの物理システムの良い近似となります。

### 1.2 入力と出力の関係

システムは、入力 $x(t)$ を受け取り、出力 $y(t)$ を生成する変換として表現できます：

$$y(t) = T\{x(t)\}$$

ここで、$T\{\cdot\}$ はシステムを表す演算子です。

**例**:
- **RC回路**: 入力電圧 $v_{in}(t)$ → 出力電圧 $v_{out}(t)$
- **質量-ばね-ダンパ系**: 入力力 $f(t)$ → 変位 $x(t)$
- **温度制御システム**: 入力熱量 $q(t)$ → 温度 $T(t)$

## 2. 線形性（重ね合わせの原理） (Linearity - Superposition Principle)

### 2.1 線形性とは

**線形性**は、システムが以下の2つの条件を満たすことを意味します：

**1. 加法性 (Additivity)**:
$$T\{x_1(t) + x_2(t)\} = T\{x_1(t)\} + T\{x_2(t)\}$$

**2. 同次性 (Homogeneity)**:
$$T\{ax(t)\} = aT\{x(t)\}$$

この2つを合わせると、**重ね合わせの原理 (Superposition Principle)** が得られます：

$$T\{ax_1(t) + bx_2(t)\} = aT\{x_1(t)\} + bT\{x_2(t)\}$$

ここで、$a$ と $b$ は任意の定数です。

### 2.2 線形性の簡単な例

**線形なシステムの例**:
- $y(t) = 3x(t)$ （定数倍）
- $y(t) = \frac{dx(t)}{dt}$ （微分）
- $y(t) = \int_0^t x(\tau)d\tau$ （積分）

これらのシステムに $x(t) = 2x_1(t) + 3x_2(t)$ を入力すると、出力は $y(t) = 2y_1(t) + 3y_2(t)$ となります。

**非線形なシステムの例**:
- $y(t) = x^2(t)$ （2乗）
- $y(t) = \sin(x(t))$ （三角関数）
- $y(t) = x(t) + 5$ （定数項がある）

**具体例**: $y(t) = 2x(t)$ のシステムに、$x_1(t) = 1$ と $x_2(t) = 2$ を入力する場合を考えます。

- $y_1(t) = 2 \times 1 = 2$
- $y_2(t) = 2 \times 2 = 4$
- $x(t) = 3x_1(t) + 4x_2(t) = 3 \times 1 + 4 \times 2 = 11$ を入力すると
- $y(t) = 2 \times 11 = 22$
- 確認: $3y_1(t) + 4y_2(t) = 3 \times 2 + 4 \times 4 = 22$ ✓

## 3. 時不変性 (Time Invariance)

### 3.1 時不変性とは

**時不変性**は、入力信号を時間的にシフトすると、出力信号も同じ時間だけシフトすることを意味します。

数式で表すと、入力 $x(t)$ に対する出力が $y(t) = T\{x(t)\}$ のとき、入力を $\tau$ だけ遅らせた $x(t - \tau)$ に対する出力は $y(t - \tau)$ となります：

$$T\{x(t - \tau)\} = y(t - \tau)$$

**直感的な理解**: システムの特性が時間によって変化しないということです。今日測定した応答と、1ヶ月後に同じ入力で測定した応答が同じになります。

### 3.2 時不変性の例

**時不変なシステムの例**:
- 理想的な電気回路（抵抗、コンデンサ、インダクタが劣化しない）
- 理想的な機械システム（摩擦係数が一定）

**時変（時不変でない）システムの例**:
- バッテリーの放電（時間とともに性能が低下）
- 経年劣化する部品を含むシステム
- 時間によって係数が変化する $y(t) = t \cdot x(t)$

**具体例**: $y(t) = 3x(t)$ のシステムを考えます。

- $x(t) = \sin(t)$ を入力すると、$y(t) = 3\sin(t)$
- $x(t - 2) = \sin(t - 2)$ を入力すると、$y(t) = 3\sin(t - 2)$
- これは $y(t - 2) = 3\sin((t - 2)) = 3\sin(t - 2)$ と一致します ✓

## 4. インパルス応答入門 (Introduction to Impulse Response)

### 4.1 インパルス信号とは

**インパルス信号 (impulse signal)** またはディラックのデルタ関数 $\delta(t)$ は、時刻 $t = 0$ で無限大の値を持ち、それ以外では0である理想的な信号です：

$$\delta(t) = \begin{cases}
\infty & t = 0 \\
0 & t \neq 0
\end{cases}$$

ただし、全体の面積（積分）は1です：

$$\int_{-\infty}^{\infty} \delta(t)dt = 1$$

**直感的な理解**: 非常に短い時間だけ非常に強い刺激を与える信号です。例えば、ハンマーで叩く、瞬間的な電圧パルス、などのイメージです。

### 4.2 インパルス応答とは

**インパルス応答 (impulse response)** $h(t)$ は、システムにインパルス信号 $\delta(t)$ を入力したときの出力です：

$$h(t) = T\{\delta(t)\}$$

### 4.3 なぜインパルス応答が重要か

インパルス応答は、LTIシステムを**完全に特徴づける**ことができます。つまり、インパルス応答が分かれば、**任意の入力に対する出力を計算できる**のです！

これは、LTIシステムの重要な性質で、次の中級編で学ぶ**畳み込み積分 (convolution integral)** によって実現されます：

$$y(t) = \int_{-\infty}^{\infty} h(\tau)x(t-\tau)d\tau = h(t) * x(t)$$

**インパルス応答の例**:
- **RC回路**: $h(t) = \frac{1}{RC}e^{-t/RC}u(t)$ （指数的に減衰）
- **質量-ばね-ダンパ系**: 振動しながら減衰する応答

ここで、$u(t)$ は単位ステップ関数（$t \geq 0$ で1、$t < 0$ で0）です。

### 4.4 ラプラス変換との関係

第2章で学んだラプラス変換を使うと、インパルス応答 $h(t)$ のラプラス変換が**伝達関数 (transfer function)** $H(s)$ になります：

$$H(s) = \mathcal{L}\{h(t)\} = \int_0^\infty h(t)e^{-st}dt$$

伝達関数は、$s$領域でのシステムの入出力関係を表します：

$$Y(s) = H(s)X(s)$$

これにより、時間領域の複雑な畳み込み積分が、$s$領域では単純な掛け算になります！

## まとめ

この初級編では、以下を学びました：

- **LTIシステム**は、線形性と時不変性を持つシステムである
- **線形性（重ね合わせの原理）**：$T\{ax_1 + bx_2\} = aT\{x_1\} + bT\{x_2\}$
