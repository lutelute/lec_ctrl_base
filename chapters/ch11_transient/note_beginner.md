# 第11章 制御系の過渡特性 - 初級編

## 1. 過渡応答とは何か

### 1.1 過渡応答の基本概念

**過渡応答 (Transient Response)** は、制御系に入力が加わったときに、出力が定常状態に落ち着くまでの間の応答のことです。制御系の性能を評価する上で最も重要な特性の一つです。

システムに入力信号が与えられると、出力は次のような経過をたどります：

1. **過渡状態 (Transient State)**: 出力が変化している期間
2. **定常状態 (Steady State)**: 出力が一定値または周期的な値に落ち着いた状態

> **注意**: 実際の制御系では、過渡応答の性能が最終的なシステムの品質を決定します。応答が速すぎると振動が発生し、遅すぎると制御の目的を果たせません。

**日常生活での例**:
- エアコンの温度調整：設定温度に到達するまでの過程
- 自動車の速度制御：目標速度に達するまでの加速過程
- ロボットアームの位置決め：目標位置に移動する過程

## 2. ステップ応答の基本

### 2.1 ステップ入力とは

**ステップ入力 (Step Input)** は、時刻 $t=0$ で突然に一定値に変化する入力信号です。数学的には次のように表されます：

$$u(t) = \begin{cases}
0 & (t < 0) \\
A & (t \geq 0)
\end{cases}$$

ここで、$A$ はステップの大きさ（振幅）です。$A=1$ の場合を**単位ステップ入力 (Unit Step Input)** と呼びます。

### 2.2 なぜステップ応答を使うのか

ステップ応答は制御系の性能評価に広く用いられます。その理由は：

1. **実現が容易**: 実際のシステムで実験しやすい
2. **情報が豊富**: 一回の試験で多くの性能指標が得られる
3. **標準化**: 異なるシステム間での性能比較が可能

**例**: 目標値を20℃から25℃に変更したときの室温の変化を観察することで、エアコンシステムの性能を評価できます。

## 3. 1次遅れ系の応答

### 3.1 1次遅れ系の伝達関数

最も基本的な制御系である**1次遅れ系 (First-Order System)** の伝達関数は次の形で表されます：

$$G(s) = \frac{K}{Ts + 1}$$

ここで：
- $K$: ゲイン (Gain) - 定常状態での出力の大きさ
- $T$: 時定数 (Time Constant) - 応答速度を決定する定数
- $s$: ラプラス変数

### 3.2 1次遅れ系のステップ応答

単位ステップ入力 $u(t) = 1$ に対する1次遅れ系の応答は：

$$y(t) = K\left(1 - e^{-\frac{t}{T}}\right) \quad (t \geq 0)$$

この式から、以下のことがわかります：

- $t = 0$ のとき: $y(0) = 0$（初期値）
- $t \to \infty$ のとき: $y(\infty) = K$（最終値）
- 応答は指数関数的に最終値に近づく

**特徴**:
- オーバーシュート（行き過ぎ）がない
- 単調増加する
- 理論的には無限時間で最終値に到達

### 3.3 具体例：RC回路

電気回路のRC回路は典型的な1次遅れ系です。

**回路構成**:
- 抵抗 $R$ とコンデンサ $C$ の直列接続
- 入力: 電圧 $v_{in}(t)$
- 出力: コンデンサの電圧 $v_C(t)$

伝達関数：

$$G(s) = \frac{1}{RCs + 1}$$

ここで、時定数 $T = RC$ です。

**数値例**: $R = 1\text{k}\Omega$, $C = 1\mu\text{F}$ のとき、$T = 1\text{ms}$ となります。

## 4. 時定数の概念

### 4.1 時定数とは

**時定数 (Time Constant)** $T$ は、1次遅れ系の応答速度を特徴づける最も重要なパラメータです。

時刻 $t = T$ における出力の値は：

$$y(T) = K\left(1 - e^{-1}\right) = K \times 0.632 \approx 0.632K$$

つまり、**時定数は最終値の約63.2%に到達する時間**を表します。

### 4.2 時定数と整定時間

実用上、出力が最終値の一定範囲内に入ることを「整定した」と判断します：

- **2%整定時間**: 最終値の±2%以内に入る時間 → $t \approx 4T$
- **5%整定時間**: 最終値の±5%以内に入る時間 → $t \approx 3T$

**計算例**:

$t = 4T$ のとき：
$$y(4T) = K(1 - e^{-4}) = K \times 0.982 = 98.2\%K$$

つまり、誤差は約1.8%となり、2%以内に収まります。

### 4.3 時定数の物理的意味

時定数は様々な物理システムで現れます：

| システム | 時定数の意味 | 例 |
|---------|------------|-----|
| 熱システム | 熱容量と熱抵抗の積 | 建物の温度変化 |
| 電気回路 | $RC$ または $L/R$ | フィルタ回路 |
| 機械系 | 慣性と摩擦の比 | モータの速度応答 |
| 化学反応 | 反応速度定数の逆数 | 濃度変化 |

> **重要**: 時定数が小さいほど、システムは速く応答します。

## 5. 特性値の紹介

### 5.1 過渡応答の評価指標

制御系の性能を定量的に評価するために、いくつかの**特性値 (Characteristic Values)** を使用します。主なものは：

1. **立ち上がり時間 (Rise Time)** $T_r$
2. **ピーク時間 (Peak Time)** $T_p$
3. **整定時間 (Settling Time)** $T_s$
4. **最大オーバーシュート (Maximum Overshoot)** $M_p$

### 5.2 立ち上がり時間 $T_r$

**立ち上がり時間** は、出力が最終値の10%から90%に到達するまでの時間です。

$$T_r = \text{出力が最終値の10%→90%に達する時間}$$

- システムの応答速度の指標
- 小さいほど速く応答する

**1次遅れ系の場合**:

$$T_r \approx 2.2T$$

### 5.3 整定時間 $T_s$

**整定時間** は、出力が最終値の一定範囲（通常±2%または±5%）内に収まる時間です。

$$T_s = \text{出力が最終値の±2\%以内に収まる時間}$$

**1次遅れ系の場合（2%基準）**:

$$T_s \approx 4T$$

### 5.4 オーバーシュート（予告）

1次遅れ系にはオーバーシュート（行き過ぎ）は発生しませんが、**2次系 (Second-Order System)** では重要な指標となります。

**オーバーシュート** $M_p$ は、出力が最終値を超えて振動する程度を表します：

$$M_p = \frac{y_{max} - y_{steady}}{y_{steady}} \times 100\%$$

中級編では、2次系の応答とオーバーシュートの詳細について学習します。

## 6. 実践例：温度制御システム

### 6.1 問題設定

部屋の温度制御システムを考えます。ヒーターの出力を1kWに設定したとき、室温が20℃から25℃まで上昇する過程を観察します。

**測定結果**:
- 初期温度: 20℃
- 最終温度: 25℃
- 3分後の温度: 23.16℃ (最終値の63.2%)

### 6.2 時定数の推定

最終値の63.2%に到達する時間が時定数なので：

$$T = 3\text{分} = 180\text{秒}$$

### 6.3 整定時間の予測

2%整定時間は：

$$T_s = 4T = 4 \times 180 = 720\text{秒} = 12\text{分}$$

したがって、室温が24.9℃～25.1℃の範囲に収まるまで約12分かかると予測できます。

### 6.4 応答の数式表現

この温度制御システムのステップ応答は：

$$T(t) = 20 + 5\left(1 - e^{-\frac{t}{180}}\right) \quad [\text{℃}]$$

ここで：
- 初期値: 20℃
- 温度変化: 5℃
- 時定数: 180秒

## 7. 練習問題

### 問題1：時定数の計算

伝達関数 $G(s) = \frac{5}{2s + 1}$ の1次遅れ系について：

(a) 時定数 $T$ を求めよ
(b) 2%整定時間を求めよ
(c) 単位ステップ入力に対する応答 $y(t)$ を求めよ

**解答**:

(a) 伝達関数を標準形 $\frac{K}{Ts+1}$ に変形：

$$G(s) = \frac{5}{2s + 1} = \frac{5}{2(s + 0.5)} = \frac{2.5}{s + 0.5} = \frac{5}{2s+1}$$

標準形に書き直すと：

$$G(s) = \frac{5}{2(s + 1/2)} = \frac{5/2}{s + 1/2}$$

分母を $Ts+1$ の形にするため：

$$G(s) = \frac{5}{2s+1} \quad \therefore T = 2, K = 5$$

時定数 $T = 2$ [秒]

(b) 2%整定時間:

$$T_s = 4T = 4 \times 2 = 8\text{秒}$$

(c) ステップ応答:

$$y(t) = K(1 - e^{-t/T}) = 5(1 - e^{-t/2}) \quad (t \geq 0)$$

### 問題2：応答速度の比較

2つの1次遅れ系 A と B があります：

- システムA: $G_A(s) = \frac{1}{s + 1}$
- システムB: $G_B(s) = \frac{1}{0.5s + 1}$

どちらのシステムが速く応答しますか？理由を説明してください。

**解答**:

システムAの時定数: $T_A = 1$ [秒]
システムBの時定数: $T_B = 0.5$ [秒]

$T_B < T_A$ なので、**システムBの方が速く応答します**。

理由：時定数が小さいほど、システムは速く最終値に到達するためです。

## まとめ

この初級編では、以下を学びました：

1. **過渡応答の概念**: 制御系が入力に対して定常状態に到達するまでの応答
2. **ステップ応答**: 制御系の性能評価に最も広く使われる試験方法
3. **1次遅れ系**: 最も基本的な動的システム、伝達関数 $G(s) = K/(Ts+1)$
4. **時定数**: 応答速度を特徴づける重要なパラメータ（最終値の63.2%に到達する時間）
5. **基本的な特性値**: 立ち上がり時間、整定時間など、性能を定量評価する指標

**重要ポイント**:
- 時定数が小さい → 応答が速い
- 1次遅れ系はオーバーシュートなし
- 整定時間 ≈ 4T（2%基準）

次の中級編では、**2次系の応答特性**について学習します。特に、減衰比と固有振動数の概念、オーバーシュートの発生メカニズム、そして詳細な特性値（$T_r$, $T_p$, $T_s$, $M_p$）の計算方法を学びます。
