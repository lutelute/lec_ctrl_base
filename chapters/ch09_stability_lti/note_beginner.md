# 第9章 LTIシステムの安定性 - 初級編

## 1. 安定性とは何か

### 1.1 日常生活での安定性

私たちの身の回りには、「安定」と「不安定」の概念がたくさんあります：

**例1: ボールと谷**
- **安定な状態**：谷の底に置かれたボールは、少し押しても元の位置に戻ります
- **不安定な状態**：山の頂上に置かれたボールは、少し押すとどんどん転がり落ちていきます
- **臨界安定な状態**：平らな面に置かれたボールは、押すと新しい位置で止まりますが、元には戻りません

**例2: 振り子**
- **安定**：振り子は揺らしても、いずれ静止位置に戻ります
- **不安定**：逆さまの振り子は、少しでも傾くと倒れてしまいます

これらの例から分かるように、**安定性 (stability)** とは「外部からの影響を受けても、システムが適切に振る舞い続ける性質」のことです。

### 1.2 制御システムにおける安定性の重要性

制御システムでは、安定性は最も重要な性質です。なぜなら：

1. **安全性**：不安定なシステムは制御不能になり、危険な状態を引き起こす可能性があります
   - 例：航空機の自動操縦システムが不安定だと墜落の危険があります

2. **性能**：安定なシステムでなければ、目標値に追従することができません
   - 例：温度調節器が不安定だと、温度が振動し続けてしまいます

3. **設計の前提**：他の制御性能（応答速度、精度など）を議論する前に、まず安定性を確保する必要があります

> **重要**: システム設計では、「安定性の確保」が最優先事項です。どんなに高性能な制御器を設計しても、不安定であれば実用できません。

## 2. BIBO安定性の直感的理解

### 2.1 BIBO安定性とは

**BIBO安定性 (BIBO Stability: Bounded-Input Bounded-Output Stability)** は、「有界入力有界出力安定性」と呼ばれ、最も基本的な安定性の概念です。

**簡単に言うと**：
- 「有界な（上限のある）入力」を加えたとき
- 「出力も有界に（上限内に）収まる」
- このような性質を持つシステムを「BIBO安定」と呼びます

### 2.2 有界とは何か

**有界 (bounded)** とは、「ある範囲内に収まっている」という意味です。

**例**：
- $|u(t)| \leq 10$ のとき、入力 $u(t)$ は有界（最大値が10）
- $|y(t)| < \infty$ のとき、出力 $y(t)$ は有界（無限大にならない）

### 2.3 BIBO安定・不安定・臨界安定の例

#### ケース1: BIBO安定なシステム

入力として $u(t) = \sin(t)$ （振幅1の正弦波）を加えたとき：

```
入力: u(t) = sin(t)  ← 有界（-1 ≤ u ≤ 1）
        ↓
    システム
        ↓
出力: y(t) = 0.5 sin(t) ← 有界（-0.5 ≤ y ≤ 0.5）
```

出力が有界に収まるので、このシステムは **BIBO安定** です。

#### ケース2: BIBO不安定なシステム

同じ入力 $u(t) = \sin(t)$ を加えたとき：

```
入力: u(t) = sin(t)  ← 有界
        ↓
    システム
        ↓
出力: y(t) = t・sin(t) ← 時間とともに増大（無限大に発散）
```

出力が時間とともに増大し無限大に向かうので、このシステムは **BIBO不安定** です。

#### ケース3: 臨界安定（限界安定）なシステム

入力 $u(t) = 0$（入力なし）でも、出力が減衰せず振動し続ける：

```
入力: u(t) = 0
        ↓
    システム
        ↓
出力: y(t) = sin(ω₀t) ← 減衰しない振動
```

このようなシステムを **臨界安定 (marginally stable)** または **限界安定** と呼びます。厳密にはBIBO安定ではありませんが、発散もしません。

### 2.4 BIBO安定性の数学的表現

BIBO安定性は以下のように定義されます：

$$
|u(t)| \leq M_u < \infty \text{ のとき } |y(t)| \leq M_y < \infty
$$

つまり：
- 入力 $u(t)$ がある定数 $M_u$ より小さければ
- 出力 $y(t)$ も必ずある定数 $M_y$ より小さい

この条件を満たすシステムが **BIBO安定** です。

## 3. 極と安定性の基本関係

### 3.1 複素平面と極の位置

LTIシステムの安定性は、**伝達関数の極 (poles)** の位置によって決まります。

**複素平面 (complex plane)** は、以下のように分けられます：

```
        虚軸 (jω)
          ↑
          |
    左半平面 | 右半平面
  (安定)  | (不安定)
          |
  --------+-------→ 実軸 (σ)
          |
          |
```

- **左半平面 (Left Half Plane, LHP)**：実部が負（$\text{Re}(s) < 0$）
- **右半平面 (Right Half Plane, RHP)**：実部が正（$\text{Re}(s) > 0$）
- **虚軸 (Imaginary Axis)**：実部がゼロ（$\text{Re}(s) = 0$）

### 3.2 極の位置と安定性の関係

**重要な規則**：

1. **すべての極が左半平面にある** → **安定**
   - 極の実部 < 0
   - 出力は時間とともに減衰する

2. **1つでも極が右半平面にある** → **不安定**
   - 極の実部 > 0
   - 出力は時間とともに増大する

3. **極が虚軸上にある**（かつ他の極はすべて左半平面）→ **臨界安定**
   - 極の実部 = 0
   - 出力は減衰も増大もせず、振動し続ける

### 3.3 具体例で理解する

#### 例1: 安定なシステム

伝達関数：
$$
G(s) = \frac{1}{s + 2}
$$

- 極：$s = -2$（左半平面）
- ステップ応答：$y(t) = 1 - e^{-2t}$ → $t \to \infty$ で $y \to 1$（収束）
- **判定：安定**

#### 例2: 不安定なシステム

伝達関数：
$$
G(s) = \frac{1}{s - 2}
$$

- 極：$s = 2$（右半平面）
- ステップ応答：$y(t) = e^{2t}$ → $t \to \infty$ で $y \to \infty$（発散）
- **判定：不安定**

#### 例3: 臨界安定なシステム

伝達関数：
$$
G(s) = \frac{1}{s^2 + 4}
$$

- 極：$s = \pm 2j$（虚軸上）
- ステップ応答：$y(t) = \frac{1}{4}(1 - \cos(2t))$（減衰しない振動）
- **判定：臨界安定**

### 3.4 なぜ極の位置が安定性を決めるのか？

極 $s = s_0$ に対応する時間応答の項は $e^{s_0 t}$ の形になります：

- **$s_0$ が負**（左半平面）：$e^{s_0 t} = e^{-|s_0|t}$ → 時間とともに減衰
- **$s_0$ が正**（右半平面）：$e^{s_0 t}$ → 時間とともに増大
- **$s_0$ が純虚数**（虚軸上）：$e^{j\omega t}$ → 振動し続ける

このように、極の実部の符号が、時間応答の減衰・増大・振動を決定します。

## 4. 視覚的なまとめ

複素平面上で極の位置と安定性の関係を整理します：

```
複素平面

    jω
     ↑
     |  × (極)
     |      減衰振動
左半平面|    ↘
安定  |     ↓ (時間応答が減衰)
     |
-----+-----→ σ
     |     ↑ (時間応答が増大)
     |    ↗
  右半平面|  × (極)
  不安定|    発散

虚軸上: × 減衰しない振動（臨界安定）
```

**覚え方**：
- **左は安心（安定）**：極が左半平面にあれば安定
- **右は危険（不安定）**：極が右半平面にあれば不安定
- **真ん中（虚軸）は微妙（臨界安定）**：減衰も発散もしない
