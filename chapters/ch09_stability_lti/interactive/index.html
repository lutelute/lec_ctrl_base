第9章 LTIシステムの安定性

Interactive Learning Tool for Stability Analysis

極配置と安定性 (Pole Placement and Stability)

使い方: 極を手動で入力するか、複素平面上をクリックして極を配置してください。 虚部が0でない極は自動的に共役極も追加されます。
安定性: 全ての極が左半平面（実部 < 0）にあれば安定、右半平面（実部 > 0）に極があれば不安定です。

極の入力

配置された極:
安定 (Stable)

複素平面 (Complex Plane)

ステップ応答 (Step Response)

ラウス表計算機 (Routh Table Calculator)

使い方: 特性多項式 \(a_n s^n + a_{n-1} s^{n-1} + \cdots + a_1 s + a_0 = 0\) の係数を入力してください。 第1列の符号変化の回数が不安定極の数を表します。

特性多項式の次数

係数入力 (最高次から順に)

時間応答シミュレーション (Time Response Simulation)

使い方: システムの極を設定し、インパルス応答またはステップ応答を観察してください。 極の位置を変更すると、応答がどのように変化するかを確認できます。
