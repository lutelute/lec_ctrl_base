# 第2章 複素数とラプラス変換 - 初級編

## 1. 複素数入門 (Introduction to Complex Numbers)

### 1.1 複素数とは何か

**複素数 (complex number)** は、実数を拡張した数の体系です。複素数は以下の形式で表されます：

$$z = a + jb$$

ここで：
- $a$ は**実部 (real part)** と呼ばれます
- $b$ は**虚部 (imaginary part)** と呼ばれます
- $j$ は**虚数単位 (imaginary unit)** で、$j^2 = -1$ を満たします

> **注意**: 数学では虚数単位に $i$ を使いますが、電気工学や制御工学では電流を表す $i$ との混同を避けるため、$j$ を使用します。

**例**:
- $z_1 = 3 + 4j$ (実部: 3, 虚部: 4)
- $z_2 = -2 + 5j$ (実部: -2, 虚部: 5)
- $z_3 = 7$ (実部: 7, 虚部: 0、実数)
- $z_4 = 2j$ (実部: 0, 虚部: 2、純虚数)

### 1.2 複素平面 (Complex Plane)

複素数は**複素平面**上の点として視覚化できます：

- **横軸（実軸）**: 実部を表す
- **縦軸（虚軸）**: 虚部を表す

複素数 $z = a + jb$ は、座標 $(a, b)$ の点として表現されます。

### 1.3 基本的な加減算

複素数の加減算は、実部と虚部をそれぞれ計算します。

**加算**:
$$(a + jb) + (c + jd) = (a + c) + j(b + d)$$

**減算**:
$$(a + jb) - (c + jd) = (a - c) + j(b - d)$$

**計算例**:
- $(3 + 4j) + (1 + 2j) = 4 + 6j$
- $(5 + 3j) - (2 + 1j) = 3 + 2j$

## 2. ラプラス変換入門 (Introduction to Laplace Transform)

### 2.1 ラプラス変換とは何か

**ラプラス変換 (Laplace Transform)** は、時間領域の関数 $f(t)$ を、複素数領域（$s$領域）の関数 $F(s)$ に変換する数学的操作です：

$$F(s) = \mathcal{L}\{f(t)\} = \int_0^\infty f(t)e^{-st}dt$$

ここで、$s$ は複素数 $s = \sigma + j\omega$ です。

**直感的な理解**:
- 時間領域の微分方程式を、$s$領域の代数方程式に変換します
- 複雑な計算が簡単になります
- 解を求めた後、**逆ラプラス変換**で時間領域に戻します

### 2.2 なぜ制御工学でラプラス変換を使うのか

制御システムの解析では、以下の理由でラプラス変換が非常に有用です：

1. **微分方程式が代数方程式になる**: システムの動的な振る舞いを表す微分方程式を、扱いやすい代数方程式に変換できます

2. **初期条件の扱いが容易**: システムの初期状態を自然に組み込むことができます

3. **伝達関数の導出**: 入力と出力の関係を明確に表現できます

4. **周波数応答の解析**: システムの周波数特性を理解できます

### 2.3 基本的な変換表

よく使われる関数のラプラス変換：

| 時間領域 $f(t)$ | ラプラス領域 $F(s)$ | 名称 |
|----------------|-------------------|------|
| $1$ (単位ステップ) | $\frac{1}{s}$ | ステップ関数 |
| $e^{-at}$ | $\frac{1}{s+a}$ | 指数関数 |
| $t$ | $\frac{1}{s^2}$ | ランプ関数 |
| $\sin(\omega t)$ | $\frac{\omega}{s^2 + \omega^2}$ | 正弦関数 |
| $\cos(\omega t)$ | $\frac{s}{s^2 + \omega^2}$ | 余弦関数 |

**例**: 指数関数 $f(t) = e^{-2t}$ のラプラス変換は $F(s) = \frac{1}{s+2}$ です。
