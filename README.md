# フィードバック制御 - インタラクティブ学習コース

## コース概要

本コースは、フィードバック制御（Feedback Control）の基礎から応用までを体系的に学ぶための講義シリーズです。制御工学は、エアコンの温度調節から自動運転車、ロボット、航空機の姿勢制御まで、現代社会のあらゆる場面で活用されている重要な技術分野です。

本コースでは、15回の講義を通じて、制御システムの解析と設計に必要な理論と実践的なスキルを習得します。各章には、初級・中級・上級の3レベルの解説と、インタラクティブな可視化ツールが用意されており、自分のペースで理解を深めることができます。

---

## 学習目標

本コースを修了すると、以下の能力を身につけることができます：

### 基礎的な理解
- 制御システムの基本概念（フィードバック、フィードフォワード）を理解する
- 複素数とラプラス変換を用いてシステムを数学的に表現できる
- 線形時不変（LTI）システムの性質を理解し、応用できる

### システム解析
- 伝達関数を用いてシステムの入出力関係を記述できる
- 周波数応答（ボード線図、ナイキスト線図）を解析できる
- 状態空間表現を用いてシステムを記述できる

### 安定性評価
- BIBO安定性の概念を理解し、システムの安定性を判別できる
- ラウス・フルビッツの安定判別法を適用できる
- ゲイン余裕と位相余裕を用いて安定性の度合いを評価できる

### 性能評価
- 過渡応答特性（立ち上がり時間、整定時間、オーバーシュート）を計算できる
- 定常偏差を計算し、システムの型を判別できる
- 時間領域と周波数領域の特性の関係を理解できる

### 制御系設計
- P制御、PI制御、PID制御の原理を理解し、設計できる
- 設計仕様を満たすコントローラパラメータを決定できる
- 実際の問題に制御理論を適用できる

## 各章の学習目標

### 第1章：制御工学の全体像
- 制御とは何かを説明できる
- フィードバック制御の基本的な仕組みを理解する
- 制御システムの4つの構成要素（制御対象、コントローラ、センサ、アクチュエータ）を説明できる
- 制御の3つの目的（安定化、目標値追従、外乱抑制）を理解する

### 第2章：複素数とラプラス変換
- 複素数の基本演算（加減乗除、極座標表現）ができる
- ラプラス変換の意味と目的を説明できる
- 基本的な関数のラプラス変換・逆ラプラス変換ができる
- 微分方程式をラプラス変換で解ける

### 第3章：LTIシステムの表現
- 線形性（重ね合わせの原理）を説明できる
- 時不変性の意味を理解する
- インパルス応答の概念を説明できる
- 畳み込み積分を用いて任意入力の出力を計算できる

### 第4章：伝達関数
- 伝達関数の定義と意味を説明できる
- 1次遅れ系の伝達関数とステップ応答を導出できる
- 極と零点の概念を理解し、安定性との関係を説明できる
- ブロック線図の等価変換ができる

### 第5章：周波数伝達関数
- 周波数応答の意味を説明できる
- 伝達関数から周波数伝達関数を求められる
- ゲインと位相を計算できる
- 1次遅れ系の周波数特性を説明できる

### 第6章：状態空間表現
- 状態変数の概念を理解する
- 状態方程式と出力方程式を書ける
- 物理システムから状態空間表現を導出できる
- 伝達関数表現との関係を説明できる

### 第7章：中間試験対策
- 逆ラプラス変換を部分分数展開を用いて計算できる
- インパルス応答と伝達関数の関係を説明できる
- フィードバック系の閉ループ伝達関数を導出できる

### 第8章：FB制御とFF制御
- フィードフォワード制御の特徴と適用例を説明できる
- フィードバック制御の利点と欠点を理解する
- 開ループ制御と閉ループ制御の違いを説明できる
- 2自由度制御の概念を理解する

### 第9章：LTIシステムの安定性
- BIBO安定性の定義を説明できる
- 極の位置と安定性の関係を理解する
- 複素平面上での極配置から安定性を判別できる
- 臨界安定（限界安定）の概念を理解する

### 第10章：FBシステムの安定性
- 特性方程式の意味を説明できる
- ラウス配列を作成し、安定性を判別できる
- フィードバックゲインと安定性の関係を理解する
- 安定限界ゲインを計算できる

### 第11章：制御系の過渡特性
- ステップ応答の特性値（立ち上がり時間、整定時間、オーバーシュート）を説明できる
- 時定数の意味と求め方を理解する
- 1次系のステップ応答を計算できる
- 2次系の減衰比と固有振動数の影響を理解する

### 第12章：制御系の定常特性
- 定常偏差の定義と意味を説明できる
- 最終値定理を用いて定常偏差を計算できる
- システムの型と定常偏差の関係を理解する
- ステップ入力、ランプ入力に対する定常偏差を求められる

### 第13章：制御系の周波数特性
- ボード線図を描き、読み取れる
- ナイキスト線図の意味を理解する
- ゲイン余裕と位相余裕を定義し、計算できる
- 周波数特性と時間応答の関係を説明できる

### 第14章：制御系の設計
- P制御の原理と特性を説明できる
- 比例ゲインが応答に与える影響を理解する
- P制御の限界（定常偏差）を説明できる
- PID制御の各要素の役割を理解する

### 第15章：期末試験対策
- 講義全体の内容を総合的に理解している
- 典型的な問題パターンに対応できる
- 計算ミスを防ぐためのチェック方法を身につける
- 考察問題に対して論理的に解答できる
